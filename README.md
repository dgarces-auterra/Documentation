# Auterra - Research Documentation

This repository contains the documentation for the various research projects carried out at Auterra. Here you will find reports, analyses, results, and relevant resources for each line of research developed.

## Structure

- Each folder corresponds to a specific research or project.
- Inside each folder, you will find the associated documentation, such as reports, articles, data, and additional resources.

## Purpose

To facilitate access, consultation, and updating of the information generated in the different research projects.
